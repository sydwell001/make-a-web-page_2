Introduction to the web

- World Wide Web: is a collection of HTML documents.
- HTML stands for: Hyper Text Markup Language.
-Webpag links: Different webpages are linked or glued together using hyperlinks e.g click me.Will take you to www.webpagelink.com

Major Pieces of the Web

There are four main components of the world wide web as follow:
1. User:This is you, sitting on a computer right now.
2. Computer: This is the machine you will use that would be having a browser (e.g Internet Explorer, Firefox, Google Chrome, Safari etc.).
3. Internet: This is the actual structure that will direct the request path to the actual destination.
4. Server: This is a computer known as server that only serve information requested by the user.

Note: a protocol known as http (hyper text tranfer protocol) is used to communicate from the user's computer via web browser through the internet path to the Server'.The server communicate back with requested information.

- Tags are also used to make links to images, webpagesetc.

Creating a structured documents with HTML

1. Tree Structure: Web browser is using a tree structure to determine how to display pages.
2. Document Object Model: Is a process were a browser take all the tags in html and create elements in a dom to form a tree like structure.
3. Boximizing Process: After the dom is formed, everything on the webpage tend to be in boxes or boxes inside boxes and its box has its own class name. The tag name div is used to define a box.

Note: Boxing makes it easy to manipulate or deal with different images, text, multimedia. The more the boxes the more the webpage get crouded.

Developer Tools

There are 3 developer tools that we covered to write HTML documents:

1. Scratchpad:This gives a real time feedback of how your webpage will look; can be used for simple webpages.
2. Codepen: This takes 2-3 seconds to give feedback of how your page will look like.
3. Sublime: This need one to refresh the page, but it is a more professional tool for web developers.

Note:All these developer tools can do the same thing, write HTML code, but each has its own advantages and disadvantages.

Stylingup with CSS

1. In CSS you first write a selector.
2. A selector defines what element of the page a particular style will apply to.
Note: Selecting a html tag in CSS means CSS rules will apply to all those html tags on the page.
3. Two ways to select html tags:
- Select tags straight to allow css rules to apply to all similar tags on the page.
- Select a class name to all css rules to apply to that particular class.

Using CSS to Avoid Repetition

CSS meaning Cascading Style Sheet can be used to avoid code repetition of similar nature.
1. Inheritance is a key feature in CSS, which is the mechanism by which properties are applied not only to a specified element but also to its descendents.
2. Descendent element inherit text-related properties, but box related properties are not inherited.

Example: Inherited and Unherited properties
Inherited Properties: Color, font, letter-spacing, line-height, list-style, text-align, text-indent, text transform, visibility, white-space and words-spacing.
Uninherited Properties: Background, border, displays, height and width, margin, min and max, overflow, padding, position, text-decoration and vertical align.

The Box Model

Everything on a webpage is displayed in a box and each box consist of 4 layers as follow:
1. Content: What is displayed on a webpage.
2. Padding: Protect content by opening space around content.
3. Border: Barricade content and Padding.
4. Margin: Clears area around the border.


Figure2: The box model overview

5. Flex box: Is an attribute that is used in CSS to have a box that auto flex.
6. Box sizing:You can size the box by pixels or by percentage. The other way to size the box is to use box-sizing attribute to border box.
7. Box position:Div elements are a block element and they take the entire width of a page. Adding the display:flex; rule in CSS will override this behavior and let div elements appear next to each other.
